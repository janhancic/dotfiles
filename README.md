# My .dotfiles

Not meant for general usage, heavily tailored for my use case.

## Instalation

```bash
cd $HOME
git clone https://github.com/janhancic/dotfiles.git .dotfiles
cd .dotfiles
./install.sh
```

Then open a new shell.
